# Master-sEndProject
## Front-End

```bash
$ yarn install
Create a .env.stage.dev file in the root of the project and configure it

We use node v14.17.0 for this project
```

## Running the app

```bash
# development
$ yarn start

# Cypress
$ yarn cy:open

#storybook
$ yarn storybook

```


## Lint
```bash
# linter
$ yarn lint
```
